## 📌 Descrizione del Repository

Questo repository documenta il mio percorso di apprendimento nello **Sviluppo Backend con Java**, nell'ambito di un programma di formazione professionale finanziato da **Forma.Temp** e organizzato da **JOB PERMANENT TALENT (JPT) B.V.**

- 📅 **Periodo:** 13 Aprile → 25 Maggio 2026
- ⏱️ **Ore totali:** 120h (sessioni giornaliere, 14:00–18:00)
- 🎯 **Obiettivo:** Passare dai fondamenti di Java a un backend completo e funzionante con Spring Boot

---

## 🗺️ Roadmap del Bootcamp

Il corso è strutturato nei seguenti moduli ufficiali:

| # | Modulo | Ore | Stato |
|---|--------|-----|-------|
| 1 | Fondamenta Java + Git & GitHub | 20h | ✅ In corso |
| 2 | Oggetti in Java + Prime logiche Backend | 20h | 🔄 In arrivo |
| 3 | Le 3 Regole Fondamentali OOP | 16h | ⏳ In arrivo |
| 4 | Introduzione a Spring & Spring Boot | 20h | ⏳ In arrivo |
| 5 | Spring Data & Architettura Backend | 20h | ⏳ In arrivo |
| 6 | Progetto Finale (Backend completo) | 16h | ⏳ In arrivo |
| — | Salute e sicurezza + Diritti lavoratori | 8h | ✅ Moduli obbligatori |

> 🟡 Il bootcamp è iniziato il 13 aprile 2026 — questo repository cresce in tempo reale.

## 🧠 Argomenti Trattati

### ✅ Fondamenta Java + Versionamento
- Variabili, tipi di dati, strutture di controllo
- Metodi, array, organizzazione in package
- Workflow Git, branching, collaborazione su GitHub

### 🔄 Programmazione Orientata agli Oggetti (OOP)
- Classi e oggetti
- Incapsulamento
- Ereditarietà
- Polimorfismo

### ⏳ Spring Boot & Backend
- Architettura MVC
- Progettazione API REST
- Configurazione progetto Spring Boot
- Dependency Injection

### ⏳ Spring Data & JPA
- Integrazione database con JPA / Hibernate
- Operazioni CRUD
- Validazione e gestione degli errori
- Pattern Repository
